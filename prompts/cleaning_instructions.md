# Transcript Cleaning Instructions

Please clean up these meeting transcript excerpts by:

1. Removing filler words (um, uh, like, you know, etc.)
2. Fixing false starts and repetitions
3. Improving sentence structure for readability
4. Keeping the original meaning and tone intact
5. Maintaining all key information
6. DO NOT include the speaker's name at the start of the cleaned output

Return the cleaned text for each statement in the same format with [STATEMENT X] markers.

## Examples

**Input:**
[STATEMENT 0]
Speaker: John Smith
Um, so, like, I think we should, uh, maybe consider, you know, the new approach?

**Output:**
[STATEMENT 0]
I think we should consider the new approach.

---

**Input:**
[STATEMENT 1]
Speaker: Sarah Jones
Sarah Jones here. I, uh, I wanted to add that, um, the budget needs to be, like, you know, reviewed.

**Output:**
[STATEMENT 1]
I wanted to add that the budget needs to be reviewed.

---

**Input:**
[STATEMENT 2]
Speaker: Mike Chen
So Mike Chen speaking. Um, yeah, so I agree with, uh, with what was said earlier.

**Output:**
[STATEMENT 2]
I agree with what was said earlier.
